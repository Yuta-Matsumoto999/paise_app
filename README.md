# praise_app

## OverView
褒め褒めワークをモバイルアプリ化しました。</br>
毎日自分で設定した個数の褒めを記録します。</br>
褒める対象は、自分 or 他人 or 子どもです。対象は自由に選択でき、組み合わせることも可能です。</br>
ただし、全く同じ内容を繰り返し記録することはできません。</br>
毎日の些細な出来事や行動を記録することで自己肯定感を高めることを目的としています。</br>
